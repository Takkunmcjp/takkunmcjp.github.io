# Typing Practice (GitHub Pages)

ブラウザだけで使える タイピング練習サイト です
インストール不要で、PC・タブレット・スマホからいつでもタイピング練習ができます

ホーム: https://typing.takkunlego0916.f5.si


# 特徴
	•	完全無料・インストール不要
	•	GitHub Pages 上で動作
    •   オープンソースで安心
	•	モード切り替え（例：単語・短文・長文）
	•	ランダム問題生成
	•	スマホ対応
	•	初心者でも使いやすいUI

あなたの学習サイトとしても、デモ用プロジェクトとしても使いやすい設計です。

# 収録されているモード


**1️⃣ 英語/日本語単語練習モード**

ランダムな単語が表示され、正しくタイプすると次へ進むモード

**2️⃣ 短文タイピング**

短い文章を素早く入力する練習

3️⃣ 長文チャレンジ

実践向けの長文を一定時間内にどれだけ正確に打てるかを測定

4️⃣ スピードテスト（WPM計測）

30秒 or 60秒の制限時間で入力速度を計測。
